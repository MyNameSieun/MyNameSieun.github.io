> uuid 라이브러리를 사용하면 고유한 id 값을 생성할 수 있다.

- uuid 설치

```
yarn add uuid
```

<br>

- uuid 사용

```jsx
import { v4 as uuid } from "uuid";

const newLetter = {
  id: uuid(),
};
```
