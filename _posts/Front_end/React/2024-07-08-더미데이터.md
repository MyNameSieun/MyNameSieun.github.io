> 컴포넌트에서 import 해서 사용하기

> 아래와 같이 map()반복문을 통해 더미 데이터를 출력할 수 있다.

```js
import dummy from "../../db/data.json";

const DayList = () => {
  return (
    <ul>
      {dummy.days.map((day) => (
        <li key={day.id}>Day {day.day}</li>
      ))}
    </ul>
  );
};

export default DayList;
```
