# ▶ inline-block

![](https://velog.velcdn.com/images/sieunpark/post/e3d8577f-2f93-4f6e-93cc-83a3390e811e/image.png)

after처럼 사진과 텍스트를 나란히 두려면 `inline-block`을 사용해야 한다.

```
    <div class="d-inline-block">
      <img src="img.jpg" alt="공부하는 소녀" />
    </div>
    <div class="d-inline-block">
      <h1>공부하는 소녀</h1>
    </div>
```
