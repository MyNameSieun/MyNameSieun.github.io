# 3. Router 셋업

pages에 Detail.jsx, Home.jsx 파일을 생성하자.

그 후, 홈화면과 상세화면에 대한 라우터 설정을 하자

```js
// Router.jsx
import { BrowserRouter, Route, Routes, Navigate } from "react-router-dom";
import Home from "pages/Home";
import Detail from "pages/Detail";

const Router = () => {
  return (
    <BrowserRouter>
      <Routes>
        <Route path="/home" element={<Home />} />
        <Route path="/detail/:id" element={<Detail />} />
        <Route path="*" element={<Navigate replace to="/" />} />
      </Routes>
    </BrowserRouter>
  );
};

export default Router;
```

```js
// App.jsx
import Router from "shared/Router";

function App() {
  return <Router />;
}

export default App;
```

## 4.2.1 스타일 컴포넌트 컨벤션

프로젝트를 진행하면서 효율적으로 코드를 작성하기 위해 나만의 스타일 컴포넌트 컨벤션을 구축하는게 중요하다는 것을 깨달았다.

따라서 여러 방법을 적용하면서 나에게 적합한 컨벤션을 찾았다.

이에 대한 내용은《 Styled-components Naming Convention 》에 포스팅하였다!

<br>

### 4.2.2 발생한 문제💫

내가 하고싶은 것: 오른쪽 하단에 버튼을 위치시키기!

![](/assets/images/2024/2024-02-21-21-58-43.png)

```js
import styled from "styled-components";

function AddForm() {
  return (
    <FormContainer>
      <Title>편지를 작성해주세요.</Title>
      <form>
        <Wrapper>
          <p>닉네임</p>
          <input type="text" />
        </Wrapper>
        <Wrapper>
          <p>내용</p>
          <textarea></textarea>
        </Wrapper>
        <Wrapper>
          <AddBtn>편지 등록</AddBtn>
        </Wrapper>
      </form>
    </FormContainer>
  );
}

// 생략

const Wrapper = styled.div`
  display: flex;
  // 생략
`;
const AddBtn = styled.button`
  background-color: #1e1e1e;
  color: white;

  border-radius: 8px;
`;

export default AddForm;
```

현재 상태! 버튼을 오른쪽 하단에 위치시키고 싶어 Wrapper 스타일 컴포넌트에 `justify-content: end;`를 적용했더니 모든 요소들이 오른쪽으로 이동하는 문제 발생!

![](/assets/images/2024/2024-02-21-22-08-47.png)

<br>

그렇다면 버튼에만 따로 스타일 컴포넌트를 적용하기 위해 ButtonWrapper을 생성!

```js
<ButtonWrapper>
  <AddBtn>편지 등록</AddBtn>
</ButtonWrapper>;

const ButtonWrapper = styled.div`
  display: flex;
  justify-content: flex-end;
`;
```

원하는 위치에 버튼이 잘 정렬되었다.

![](/assets/images/2024/2024-02-21-22-14-11.png)

<br>

컴포넌트를 생성하지 말고 props를 사용하여 버튼만 `justify-content: end;`을 적용할 수 있지 않을까? 라고 생각되어 조건부 스타일링으로 구현해보았다!

```js
<Wrapper justifyContent="flex-end">
  <AddBtn>편지 등록</AddBtn>
</Wrapper>;

const Wrapper = styled.div`
  display: flex;
  justify-content: ${(props) => props.justifyContent || "flex-start"};
  /* 생략  */
`;
```

해당 컴포넌트에 justifyContent라는 속성이 전달되면 그 값을 사용하고, 속성이 없을 경우 기본값으로 'flex-start'를 사용하도록 설정해주었다.

<br>

아래와 같은 방법으로도 구현할 수 있다.

```js
<Wrapper flexEnd>
  <AddBtn>편지 등록</AddBtn>
</Wrapper>;

const Wrapper = styled.div`
  display: flex;
  justify-content: ${(props) => (props.flexEnd ? "flex-end" : "flex-start")};
  /* 생략  */
`;
```

flexEnd prop이 전달되면 justify-content: flex-end; 스타일이 적용되고, 그렇지 않으면 justify-content: flex-start; 스타일이 적용된다.
